UNIX is a operating system that is able to allow its multiple users to have multiple tasks running at the same time, which is different from other systems. It was developed in the 70s by scientist. Different version of this program exist, because it was licensed out to many companies.
Linux was developed in the 90s and is also a multitasking operating system like UNIX but is used on personal computers. It produces its user codes for them and its one of the most popular operating systems.
