One operating system function is Starting a PC or mobile device. You have to press the power button to send an electrical signal to the device. This signals the computer to check the system to make sure everything is running properly.
Shutting Down/turning off devices is an operating system function. There are many different ways of shutting down computer, there is sleep mode which puts the device in a low powered state that save all necessary work and closes everything else. Hibernate mode is the other option which saves open program information to a internal hard drive before powering down.
